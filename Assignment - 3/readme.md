
# Task List Application

## How to Run

1. Ensure you have Ruby installed on your system. You can download and install Ruby from [here](https://www.ruby-lang.org/en/downloads/).

2. Clone this repository to your local machine:

    ```bash
    git clone https://github.com/Rinil-Parmar/Ruby_Assignments.git
    ```

3. Navigate to the project directory:

    ```bash
    cd '.\Assignment - 3\'
    ```

4. Run the application:

    ```bash
    ruby Task_List_App.rb
    ```

## Usage

- Add tasks to the task list.
- Add priority tasks to the task list.
- List all tasks.
- Mark a task as complete.
- Find a task by its title.
- Find a priority task by its title.
